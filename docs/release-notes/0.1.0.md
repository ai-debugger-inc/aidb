# AI Debugger Release Notes: 0.1.0

**GitHub Release**:
[0.1.0](https://github.com/ai-debugger-inc/aidb/releases/tag/0.1.0)

## Overview

First public release of AI Debugger (AIDB) - a lightweight, language-agnostic Python API that enables AI systems to programmatically control and introspect live debugging sessions through the Debug Adapter Protocol (DAP).

## Breaking Changes

- N/A (initial release)

## New Features

**Multi-Language Debugging Support**

- Python debugging via debugpy adapter
- JavaScript/TypeScript debugging via vscode-js-debug adapter
- Java debugging via java-debug adapter

**MCP Integration**

- 12 debugging tools exposed through Model Context Protocol
- Context-aware next steps with scenario-specific guidance
- Size-optimized responses for efficient context usage

**Core Debugging Capabilities**

- Execution control (launch, attach, continue, pause)
- Step execution (into, over, out)
- Breakpoint management (line, conditional, hit count, logpoints)
- Variable inspection and evaluation
- Stack trace and thread information
- Multi-session debugging support

**Advanced Breakpoint Features**

- 7 hit condition modes (>, >=, =, \<, \<=, %, exact)
- Logpoints with template syntax
- Column-level breakpoints for minified code
- Real-time verification and state tracking

**Framework Support**

- Python: pytest, django, flask, fastapi, pyramid, asyncio, behave
- JavaScript/TypeScript: jest, mocha, node, express, typescript
- Java: JUnit, Spring, Maven, Gradle

**VS Code Integration**

- Full launch.json support with variable substitution
- Zero duplicate configuration
