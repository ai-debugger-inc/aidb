# AI Debugger Release Notes: 0.0.8

**GitHub Release**:
[0.0.8](https://github.com/ai-debugger-inc/aidb/releases/tag/0.0.8)

## Overview

MCP server rebranding and configuration cleanup.

## Enhancements

- Renamed MCP server from `aidb-debug` to `ai-debugger` for better brand consistency

## Internal Changes

- Removed deprecated `enforcement` and `priority` fields from skill-rules.json schema
- Updated all MCP client configuration examples in documentation

## Breaking Changes

- MCP server name changed from `aidb-debug` to `ai-debugger`
  - Users must update their MCP client configuration to use the new server name
  - Tool prefixes change from `mcp__aidb-debug__*` to `mcp__ai-debugger__*`
