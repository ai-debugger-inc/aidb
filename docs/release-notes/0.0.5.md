# AI Debugger Release Notes: 0.0.5

**GitHub Release**:
[0.0.5](https://github.com/ai-debugger-inc/aidb/releases/tag/0.0.5)

## Overview

CI/CD refactoring release - extracted inline Python scripts into reusable modules for adapter build workflows.

## Enhancements

- Extracted adapter build configuration logic into standalone script (`extract_build_config.py`)
- Extracted manifest generation logic into standalone script (`generate_manifest.py`)
- Reduced workflow YAML complexity by removing inline Python heredocs

## Internal Changes

- Added `.github/scripts/adapters/` directory for adapter build utilities
- Refactored `adapter-build.yaml` to use external scripts
- Refactored `release-pr.yaml` to use external scripts
- Updated version to 0.0.5 across all package modules
