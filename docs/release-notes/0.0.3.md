# AI Debugger Release Notes: 0.0.3

**GitHub Release**:
[0.0.3](https://github.com/ai-debugger-inc/aidb/releases/tag/0.0.3)

## Overview

Patch release with CI/CD improvements for more reliable release automation.

## Changes

- Added pre-commit hook to validate release notes exist on release branches
- Optimized CI to skip redundant test runs when merging release PRs to main
- Improved version sync script to prioritize release branch names over stale git tags

## Fixed

- Fixed `release-publish` workflow failing due to invalid `isLatest` field in `gh release view`
- Fixed version sync using stale git tags instead of current release branch version
- Fixed actionlint security warning by passing commit message through environment variable
