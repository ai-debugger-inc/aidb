# AI Debugger Release Notes: 0.0.2

**GitHub Release**:
[0.0.2](https://github.com/ai-debugger-inc/aidb/releases/tag/0.0.2)

## Overview

Patch release fixing package dependencies and improving CI/CD pipeline reliability.

## Features

- Language-agnostic debugging API supporting Python, JavaScript, and Java
- MCP server integration with 12 agent-optimized debugging tools
- Debug adapter management and auto-installation
- Session lifecycle management with breakpoints, stepping, and variable inspection
- VS Code launch.json configuration support

## Changes

- Moved YAML utilities from `aidb_common` to `aidb_cli` to remove pyyaml as a core dependency
- Extracted PyPI upload logic into reusable workflow for better maintainability
- Added idempotent version checking for PyPI uploads (safe workflow re-runs)

## Fixed

- Fixed `ModuleNotFoundError: No module named 'yaml'` when installing core package
- Fixed PyPI upload failures on workflow re-runs by checking version existence
