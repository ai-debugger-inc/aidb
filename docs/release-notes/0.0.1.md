# AI Debugger Release Notes: 0.0.1

**GitHub Release**:
[0.0.1](https://github.com/ai-debugger-inc/aidb/releases/tag/0.0.1)

## Overview

Initial release of AI Debugger (aidb) - a lightweight, language-agnostic Python API that enables AI systems to programmatically control and introspect live debugging sessions through the Debug Adapter Protocol (DAP).

This release validates the full CI/CD pipeline including PyPI distribution.

## Features

- Language-agnostic debugging API supporting Python, JavaScript, and Java
- MCP server integration with 12 agent-optimized debugging tools
- Debug adapter management and auto-installation
- Session lifecycle management with breakpoints, stepping, and variable inspection
- VS Code launch.json configuration support
